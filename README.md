# NexPhase Contracting Services Website

This is a responsive business website built for NexPhase Contracting Services, a construction and renovation company serving Lethbridge and nearby areas.

## Live Website

https://nexphase-contracting.netlify.app/

## Features

- Responsive design
- Mobile navigation menu
- Contact form
- Google Maps section
- WhatsApp contact button
- Services section
- Work/project showcase section

## Services Listed

- Construction
- Renovation
- Home Maintenance
- Flooring
- Painting

## Technologies Used

- HTML
- CSS
- JavaScript
- Netlify
- Formspree

## Project Purpose

The purpose of this project was to create a clean and professional website for a construction business so customers can view services, contact the company, request quotes, and access the business online.
